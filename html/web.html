This web page is my first web project. I completed the codeacademy 'Deploy a Website', 'Learn HTML & CSS: Part I', and 'Learn JavaScript' before creating it.

The page is hosted by github, and I used Amazon Web Service: Route 53 to acquire the domain name and setup a hosted zone.

The twitter bootstrap framework was used for the side-nav-bar and some of the layout. I used Visual Studio as the IDE. The fonts are from google fonts.

I tried to make sure the page works well on mobile and dektop, so if you're curious check it out on both mediums.
